# htmlPlayer
Pure HTML/JS audio player for Multimedia assignment

Features:
- Audio playback
- User file upload
- Custom UI controls (SVG buttons, styling e.t.c.)
- Volume controls
- Seek by clicking progress bar
- Mute/Unmute (interacts intelligently with volume controls)
- Current playback time
- Skip forward and backward
- Keyboard controls (space to play/pause, arrow keys to skip forward/back)
- QoL error prevention (e.g. play/pause button and progress bar are static when there is no file selected)

I've also included the song I was using while testing in case you're feeling particularly open-minded :)
